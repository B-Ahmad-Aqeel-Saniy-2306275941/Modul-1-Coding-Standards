# Modul 1 - Coding Standards

## Reflection 1

Dalam tutorial modul 1 ini, saya belajar mengenai clean code dan coding standards. Kode yang dibuat dalam aplikasi tersebut memiliki format yang konsisten sehingga mudah untuk dibaca. Penamaan function dan variabel juga dibuat agar dapat dipahami tanpa menjelaskan function tersebut. Penamaan variabel dan file name dibuat secara konsisten. Melihat kode saya sendiri, dapat dilihat bahwa terdapat bagian kode yang dapat bermasalah jika aplikasi didevelop dengan skala yang lebih besar, seperti product ID yang bernilai sebagai index dan bukan sebagai UUID.
